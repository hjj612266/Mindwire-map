# Mindwire

A mind mapping system built with pure HTML5, CSS3, and Vanilla JS — zero build tools, zero dependencies.

- Infinite canvas with dot-grid background, pan & zoom
- Drag-to-connect cards via bezier-curved SVG connections with obstacle avoidance
- Particle system with mouse repulsion and inter-particle lines
- Dark/light theme + 6 accent colors
- Multi-map management with localStorage persistence
- Express backend (port 4010) for optional file-based storage

## Quick start

```bash
start.bat
```

Or manually: `cd server && npm install && node server.js`, then open http://localhost:4010.
